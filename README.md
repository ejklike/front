#Front

### Dependancies
* npm
* yarn : `npm install yarn`

### How to run
1. 패키지 설치: `yarn install`
1. 개발 환경 테스트: `yarn start`
1. 배포판 빌드: `yarn build`

cf. 위의 명령들은 npm으로도 동일하게 실행 가능

### How to add dependancies
* dev dependancies: `yarn add [package name] -D`
* dependancies: `yarn add [package name]`
